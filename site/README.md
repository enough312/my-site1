# Интерактивная карта Лениногорска

Web-проект для номинации «Web-сайты (Web-проекты)» муниципального конкурса «Юный программист».

## Что умеет проект

- показывает интерактивную локальную карту Лениногорска на основе OpenStreetMap;
- отмечает достопримечательности, парки, памятные места и объекты нефтяной истории;
- поддерживает поиск и фильтрацию по категориям;
- открывает карточку выбранного места с адресом, годом и описанием;
- открывает подробное окно с фотографией и расширенной справкой по клику на метку;
- позволяет собрать маршрут школьной экскурсии;
- содержит мини-викторину по фактам о городе;
- сохраняет выбранный маршрут в браузере.
- поддерживает приближение, отдаление, сброс масштаба и перетаскивание карты при увеличении.
- адаптируется под экран телефона: карта, фильтры и подробные карточки перестраиваются в одну колонку.

## Как запустить

Откройте файл `index.html` в браузере. Основная карта сохранена как локальный фрагмент OpenStreetMap, фотографии сохранены в `assets/photos`, поэтому проект открывается без подгрузки внешних тайлов и не ломается при слабом интернете.

## Использованные средства

- HTML5;
- CSS3;
- JavaScript;
- локальная карта OpenStreetMap;
- локальное хранение маршрута в браузере.
