# Аннотация к web-проекту

Название проекта: «Интерактивная карта Лениногорска».
URL проекта: локальный файл `index.html`; при публикации в интернете указывается ссылка на размещенный сайт.
Назначение проекта: помочь школьникам и гостям города быстро познакомиться с важными местами Лениногорска.
Аналогичные разработки обычно показывают только список достопримечательностей или обычную карту без учебных функций.
В проекте объединены локальная карта Лениногорска, фото-метки, подробные карточки объектов, фильтры, поиск, масштабирование, маршрут и мини-викторина.
Алгоритм работы: пользователь выбирает категорию или вводит запрос, после чего на карте остаются подходящие точки.
При выборе маркера открывается подробная карточка с фотографией, описанием, адресом, годом, историей, советом для осмотра, интересным фактом и практической значимостью.
Маршрут сохраняется в браузере и отображается линией на карте.
Для запуска нужно открыть файл `index.html` в браузере; основная карта города работает без подключения к интернету.
Использованные средства: HTML5, CSS3, JavaScript, локальная подложка OpenStreetMap и локальное хранение данных в браузере.
Интерфейс адаптирован для компьютера и телефона.
Практическая значимость: проект можно использовать на классном часе, экскурсии или уроке краеведения.
Перспективы развития: добавить фотографии мест, аудиогид, больше объектов района и режим построения маршрута по времени прогулки.
